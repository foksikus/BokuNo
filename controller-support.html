Setup

  1. Download and install BokuNo core (see home page)
  2. The controller support plugin is included in the plugins folder
  3. Connect your Xbox controller before launching the game
  4. Customize button mappings by editing plugins/config_gamepad.json

Configuration

Edit config_gamepad.json to customize button mappings. Available actions:

Attack

Attack the currently selected target

UseSkill

Cast a skill (requires skill_id and level)

UseItem

Use an item from inventory (requires item_id)

NextTarget

Cycle through nearby enemies

CenterCamera

Reset camera to default view

Known Issues

  • Target Selection Icon: The target indicator works best if you manually click a monster with your mouse first
  • Setup Application: Remove ddraw.dll temporarily if you need to run opensetup.exe
  • Ground Skills: Not supported yet - planned for future update
  • Controller Compatibility: Only tested with Xbox controllers, other types may not work

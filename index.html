::bokuno

Plugin System for Ragnarok Online

Extend Your RO Experience

Bokuno is a custom plugin system that extends the capabilities of the Ragnarok Online client. Load custom plugins to add new features and enhance your gameplay.

::Download(::bokuno);

Available Plugins

Gamepad Support

Play Ragnarok Online with full gamepad support. Customizable button mappings for movement, combat, and skills.

Installation

  1. Download the BokuNo.zip package from GitHub releases
  2. Extract and copy ddraw.dll (the BokuNo core) to your Ragnarok Online root folder
  3. Copy the plugins folder to your Ragnarok Online root folder
  4. Launch Ragnarok Online - plugins will load automatically

Known Issues

  • Target Selection Icon: The target indicator works best if you manually click a monster with your mouse first
  • Setup Application: Remove ddraw.dll temporarily if you need to run opensetup.exe
  • Ground Skills: Not supported yet - planned for future update
  • Gamepad Compatibility: Only tested with Xbox Gamepads, other types may not work

::auto_login

::bokuno Plugin

Auto Login for Ragnarok Online

Automatically login into Ragnarok Online

Seamlessly log into your Ragnarok Online account without manual input. Save time and enhance your gaming experience with our Auto Login plugin.

::Download(::bokuno);

Demo

Setup

  1. ::Download and install bokuno-core (also see ::main page)
  2. The Auto Login plugin is included in the plugins folder
  3. Configure your login credentials in plugins/config_auto_login.json

Configuration

Edit config_auto_login.json to set your login credentials.

Type definitions

                    
type AccountName = string;
type Password = string;

interface AutoLoginConfig {
    username: AccountName;
    password: Password;
}

Example

                    
{
    "username": "YourAccountName",
    "password": "YourPassword"
}
